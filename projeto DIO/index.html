Títulos e subtítulos

Como funcionam: h1 é o título mais importante (tipo o nome do site), e o h6 é o menos importante. Vai do maior ao menor.

Parágrafo

Para que serve "p": Usado para escrever um texto comum.

Marca texto

Para que serve "mark": Destaca uma parte do texto, como se estivesse grifando com caneta marca-texto.

Texto pequeno

Para que serve "small": Diminui o tamanho da letra para observações ou notas de rodapé.

(Voltar)

itálico

Para que serve "italico": Deixa o texto inclinado, geralmente para destaque, citações ou palavras estrangeiras.

Sublinhado

Para que serve "u": Deixa o texto com uma linha embaixo.

Negrito com importância

Para que serve "strong": Destaca e dá importância a uma palavra ou frase.

Lista ordenada

Para que serve "ol": Criar uma sequência com números

Lista não ordenada

Para que serve "ul": Criar uma lista simples sem ordem.(lista com bolinhas)

Item da lista

(Voltar)

Para que serve "li": Vai dentro de ol ou ul para mostrar cada item.

Link

Para que serve "a": Criar um botão ou palavra clicável que leva para outro lugar.

Linha horizontal

Para que serve "hr": Separar conteúdos visualmente.

Texto embaixo(subscrito)

Para que serve "sub": Usado em fórmulas químicas ou rodapés.

H2O

Texto em cima(sobrescrito)

Para que serve "sup": Usado em potências, fórmulas matemáticas, etc.

(Voltar)

Citação longa

Para que serve "blockquote": Usado para mostrar que você está citando um texto ou fala de outra pessoa.

Altera o tipo,cor ou tamanha da fonte

O que é "font": Altera o tipo, cor ou tamanho da fonte.


hoje em dia: Evita-se usar, mas você pode usar no exercício.

Texto riscado(deletado)

Para que serve "del": Mostra que aquele texto foi removido.

Abreviação com significado

Para que serve "abbr": Quando passa o mouse, mostra o significado da sigla.
